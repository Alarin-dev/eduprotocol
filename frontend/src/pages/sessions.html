EduProtocol

It's an educational platform project based on a value-sharing system between learners and teachers using internal tokens.

The core idea was that: Anyone can offer sessions to teach skills they know.

In return for offering these sessions, they receive tokens.

Learners can use these tokens to learn skills from others.

A person who knows a skill receives a token, which allows them to learn a skill they don't know from others.
